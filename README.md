# Тестовое задание
Основные технологии:

* Vue3
* Webpack

#### Команды
```bash
# установка пакета
npm i
# сборка проекта
npm run build
# запуск на локальном сервере
npm run serve
```
---
Реализована адаптивная верстка, новости подтягиваются через Fetch запрос

Работу можно посмотреть [тут](https://akvela.github.io/app-news/)
